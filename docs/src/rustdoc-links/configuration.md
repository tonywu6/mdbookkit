# Configuration

This page lists all options for the preprocessor.

Configure options in the `[preprocessor.rustdoc-links]` table using the keys below, for
example:

```toml
[preprocessor.rustdoc-links]
rust-analyzer = "path/to/rust-analyzer --option ..."
```

For use [on the command line](standalone-usage.md), specify options via arguments, for
example:

```bash
mdbook-rustdoc-links markdown --rust-analyzer "..."
```
